# RPG Engine for Löve
An attempt on a RPG engine based on Löve2D (0.9.1).

## Current features:
- [x] Player mechanics
  - [x] Walking
  - [x] Running
  - [x] Jumping (ugly as hell by the way)
- [ ] Box mechanics
  - [x] Message box (no rich text yet)
  - [ ] Selection box
- [ ] Battle system

## How to use
Actually _don't_.
It's not ready to be used as an engine.
